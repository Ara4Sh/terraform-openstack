# terraform-openstack

OpenStack examples using terraform. 

# Usage
Source your openstack rc file
```sh
source admin.rc
```
then edit `variables.tf` with your favorite editor and follow these steps

```sh
terraform init
terraform plan
terraform apply 
```
to check the output:
```sh
terraform output
```
